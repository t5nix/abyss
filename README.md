# Abyss

**Abyss** is a customizable tool for generating tailored Linux ISO images.  
It is designed to simplify the creation of reproducible, declarative, and lightweight systems for security professionals, researchers, and developers.

---

## Features

- **Declarative Configuration**  
  Define your entire system in simple JSON files. Version control your infrastructure and reproduce identical environments across teams.

- **Pre-Built Templates**  
  Start with battle-tested templates for pentesting, forensics, malware analysis, or development. Customize to fit your exact needs.

- **Automated Builds**  
  Generate bootable ISOs in minutes, not hours. 

- **Package Management**  
  Cherry-pick from thousands of packages. Include custom scripts, configs, and tools. Remove bloat you don't need.

- **Offline Capability**  
  Build once, deploy anywhere. Perfect for air-gapped environments, training labs, or field operations without internet.

- **Live & Persistent Modes**  
  Boot as live USB for quick assessments or install with persistence. Your choice, your workflow.
